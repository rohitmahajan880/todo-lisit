# Todo List

A simple standalone Todo List app built with HTML, CSS, and JavaScript.

## Files

- `index.html` — app UI and structure
- `styles.css` — styling for layout and controls
- `script.js` — todo item creation, completion, removal, and localStorage persistence

## Features

- Add tasks
- Mark tasks complete
- Remove tasks
- Clear completed tasks
- Persist tasks in browser `localStorage`

## Usage

1. Open `index.html` in your browser.
2. Enter a task in the input field.
3. Click `Add` or press Enter.
4. Use the checkbox to mark tasks complete.
5. Click `Remove` to delete a task.
6. Click `Clear completed` to remove completed tasks.

## Notes

- The app stores tasks locally in the browser. Reloading the page keeps the current todo list.
- No build step or dependencies are required.
